# 银邻互助站 —— 面向老人社区的生活互助与关怀服务平台

> 副标题：让社区里的每一位老人，都能被及时看见和温暖回应。

## 项目背景

社区老人在日常生活中常遇到买菜代买、陪同就医、上门维修、手机操作指导、临时陪伴等需求，而家属无法时刻陪伴、社区志愿者力量分散、居委会难以实时掌握动态。银邻互助站面向社区老人、家属、志愿者和社区工作人员，提供适老物品共享、健康与便民资料、日常生活互助、临时关怀求助、志愿服务招募、社区关怀数据大屏等能力，通过实名认证、联系方式脱敏、上门服务留痕、家属授权查看等机制，搭建一个温暖、可信、适老化的社区关怀桥梁。

## 核心用户

1. **社区老人**：发布生活求助、预约陪诊、申请代买、寻找陪伴、查看活动。
2. **家属**：查看老人求助记录、关注服务状态、了解社区动态。
3. **志愿者**：响应求助、报名服务、积累志愿时长和信用积分。
4. **社区工作人员**：管理互助信息、查看数据大屏、维护安全规则。

## 功能模块

- **首页 Dashboard**：温暖宣传语、核心数据卡片（今日互助请求 / 已响应 / 活跃志愿者 / 本月陪诊代买上门关怀）、最近求助动态、温暖互助故事、防诈骗提示、志愿者服务榜、安全可信说明、一键求助入口。
- **适老物品共享**：拐杖、轮椅、助听器电池、血压计、老花镜、护理垫、便携小推车、康复训练用品等分类；支持免费借用、低价转让、爱心赠送、短期租借；保留发布、搜索、筛选、申请联系、状态展示。
- **健康与便民资料**：慢病管理小贴士、防诈骗指南、社区医保办理流程、就医挂号流程、智能手机使用教程、社区活动通知、居家安全指南；保留发布、搜索、分类筛选、收藏、查看详情、模拟上传文件。
- **老人日常生活互助**：帮忙买菜、代买药品、取送快递、维修帮忙、搬运小件、陪同散步、上门查看、手机操作指导；状态包含待响应 / 已响应 / 进行中 / 已完成 / 已取消；涉及药品仅限代买常规药品，不做医疗诊断建议。
- **临时关怀求助**：手机操作求助、陪同就医、家中维修、雨天代买、陪伴聊天、寻找物品；紧急程度分普通 / 较急 / 紧急；页面提示"如遇突发疾病、严重摔倒、意识不清等情况，请立即拨打急救电话并联系家属"。
- **志愿服务招募**：周末陪伴聊天、智能手机教学、社区义诊秩序协助、爱心送餐、楼栋安全巡访、节日慰问活动；展示报名人数、目标人数、服务时长、进度条、活动地点、活动时间。
- **社区关怀数据大屏**：各楼栋求助热度、今日服务响应情况、高频求助类型、志愿者活跃情况、重点关怀老人数量、最近实时动态；保留 Three.js 三维社区关怀网络可视化。
- **用户中心**：姓名脱敏、手机号脱敏、身份证脱敏、所属楼栋、身份类型（老人 / 家属 / 志愿者 / 社区工作人员）、互助积分、信用分、志愿时长、我的求助、我的响应、我的报名。
- **社区安全规则**：实名认证、联系方式脱敏、上门服务留痕、志愿者信用积分、异常行为举报、老人隐私保护、家属可查看授权记录、贵重物品不建议私下交易、涉及身体不适时建议联系家属或专业机构。

## 适老化体验

- 字体更大、按钮更明显、卡片间距更舒适、表单提示更清楚。
- 重要状态用明显标签，避免复杂嵌套操作。
- 暖橙、米白、浅绿、柔和蓝等亲和配色，降低视觉疲劳。
- 保留 GSAP 动画但更柔和，不炫技。
- 一键求助入口在首页和侧边栏明显位置展示。
- 防诈骗提示卡片提醒老人不要轻信陌生转账、中奖、冒充亲友等信息。
- 温暖故事模块展示社区互助小故事。

## 技术栈

- React + Vite
- Tailwind CSS（自定义温暖适老化配色）
- GSAP + @gsap/react（页面过渡、卡片 reveal、数字滚动、Modal 弹出）
- Three.js（三维社区关怀网络可视化 + OrbitControls）
- localStorage 持久化（useLocalStorage hook）
- lucide-react 图标

## 运行方式

```bash
npm install
npm run dev
```

打开浏览器访问 Vite 提示的本地地址即可。所有数据通过 localStorage 模拟持久化，刷新页面后发布的物品、求助、报名、收藏、申请记录不会丢失。

## 项目亮点

- **真实社区场景**：覆盖楼栋、活动中心、社区食堂、卫生服务中心、便民服务站等高频互助场景。
- **适老物品共享**：拐杖、轮椅、血压计、助听器电池、护理垫等适老化分类，支持借用、转让、赠送、租借和信用分门槛。
- **健康与便民资料**：慢病管理、防诈骗、医保办理、就医挂号、智能手机教程等老人高频需求资料。
- **生活互助全流程**：从待响应到已完成的完整状态流转，涉及药品时仅做代买常规药品，不做医疗诊断。
- **临时关怀求助**：紧急程度分级，页面强提示"如遇突发疾病请立即拨打 120"，不代替急救。
- **志愿服务招募**：报名人数和进度条即时变化，志愿时长展示。
- **社区关怀数据大屏**：把互助数据接入三维社区关怀网络图，展示楼栋求助热度、重点关怀老人、志愿者活跃情况。
- **实名 + 信用机制**：用户中心展示信用分、互助积分、志愿时长和互助记录。
- **隐私脱敏**：手机号、身份证号、联系方式默认脱敏显示，家属可查看授权记录。
- **localStorage 持久化**：刷新页面后发布、收藏、报名、申请记录不丢失。
- **GSAP 动画增强体验**：页面切换、数字滚动、卡片 stagger reveal、Modal 弹出、进度条、按钮微交互和 Sidebar active indicator，动画柔和适老。
- **适老化 UI**：大字号、高对比、清晰按钮、暖色配色，重要状态用明显标签。

## 安全说明

- 本平台仅做社区互助信息撮合，不涉及真实支付、真实医疗诊断、真实定位追踪。
- 涉及身体不适时建议联系家属或专业机构，紧急情况请立即拨打 120。
- 贵重物品不建议私下交易，上门服务需留痕，家属可查看授权后的服务记录。
- 所有数据均通过 localStorage 模拟持久化，不会上传到任何服务器。
